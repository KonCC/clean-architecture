# 1장 설계와 아키텍처란
## 설계와 아키텍처의 차이는 없다
<일반적인 경우>  
설계 : 저수준의 구조/결정 사항  
아키텍처 : 저수준의 세부사항과는 다른 고수준의 무언가  
- 설계와 아키텍처는 개별로 존재할 수 없으며, 서로 연결된 것이다  
고수준의 아키텍처 속에는 이미 저수준의 설계가 포함돼있다
